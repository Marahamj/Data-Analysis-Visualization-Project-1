Project Overview
This project demonstrates a complete Data Science workflow, starting from automated data procurement to deep exploratory analysis. It integrates two major components: a dynamic data acquisition pipeline using the Kaggle API and an intensive Exploratory Data Analysis (EDA) phase focused on data integrity and visualization.

Project Structure
The repository is organized into two main stages:

Data Acquisition (assiment 2 kaggle.ipynb): Automates the retrieval of large-scale datasets directly from Kaggle servers.

Data Engineering & EDA (assiment2 data 2.22.ipynb): Focuses on cleaning raw data, handling inconsistencies, and extracting statistical trends.

Core Skills & Technologies
Data Engineering: Implementing automated pipelines for data retrieval and storage.

Advanced Data Cleaning: Systematic handling of missing values, outliers, and data type optimization.

Exploratory Data Analysis (EDA): Identifying correlations and hidden patterns within complex datasets.

Visual Analytics: Creating high-impact visualizations using Seaborn and Matplotlib.

Tech Stack: Python (Pandas, NumPy), Kaggle API, Jupyter Notebooks
